RULES

Product Card:

  • Hover On:
    • There is an overlay that fades in on the image with 25% opacity, darkening the image in 0.2 seconds
    • A save folder appears in the top left corner in a capsule shape, with a folder icon on the left, and the text “save” on the right
    • A blue bookmark shaped tag (with the abbreviation of the tag in two letters, placed towards the bottom of the tag) slides down over 0.4 seconds
    • Cursor changes to finger
  • Hover Off:
    • The overlay disappears over 0.2 seconds and the image brightens to original state
    • The save capsule fades away with the overlay
    • The bookmark tag banner slides up in 0.4 seconds
    • Finger changes back to cursor
  • Click:
    • Goes to/links to a new webpage

Save Button:

  • Hover On:
    • The slightly transparent and blurred black background color of the button darkens from 60% opacity to 90% opacity (over product images that have a light background)
    • Over product images that have a dark background, the button lightens to a greyish color
  • Hover Off:
    • The button color goes back to its original state
  • Click:
    • Goes to/links to a new webpage

Tag Banner:

  • Hover On:
    • The banner slides down by 15% of its height
  • Hover Off:
    • The banner slides back up to original position, before it was hovered over
  • Click:
    • Goes to/links to a new webpage

Product Card Title:

  • Hover On:
    • The title (with the text: "Fading Echoes") is underlined, which fades in over 0.6 second
    • The pointer changes to finger
  • Hover Off:
    • The title's underline fades out over 0.6 second
    • The finger changes back to pointer
  • Click:
    • Goes to/links to a new webpage

Product Card Author:

  • Hover On:
    • The author (with the text: "Omar Eltahan") is underlined, fades in over 600ms
    • A popup with information on the author slides up from the author text, over 0.7 second, to stop above the author text
  • Hover Off:
    • The author's underline fades out over 0.6 second
    • The popup slides back down out of view

Thumbs Up and Count:

  • Hover On:
    • The pointer changes to finger
  • Hover Off:
    • The finger changes back to pointer
  • Click:
    • The thumb icon tilts up by 25 degrees and back down to original position over 0.8 second
    • The thumb icon fades from grey to blue over 0.3 second
    • Database is updated, with the counter increasing by 1
  • Click Again:
    • The thumb icon fades from blue back to grey over 0.8 second
    • Database is updated, with the counter decreasing by 1
