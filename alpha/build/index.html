TRIGGERS

Product Card

  • Hover On
  • Hover Off
  • Click

RULES

Product Card:

  • Hover On:
    • There is an overlay on the image with 30% opacity, darkening the image
    • A save folder appears in the top left corner in a capsule shape, with a folder icon on the left, and the text “save” on the right
    • A blue bookmark shaped tag (with the abbreviation of the tag in two letters, placed towards the bottom of the tag) slides down but the sliding down action is very smooth and quick so users probably don’t perceive it
    • Cursor changes to finger
  • Hover Off:
    • The overlay disappears and the image brightens to original state
    • The save capsule disappears right away
    • The bookmark tag disappears right away
    • Finger changes back to cursor
  • Click
    • Goes to/links to a new webpage
